## Honest Feedback

**3 Friction Points**:

1. **Subagent permissions ใน /awaken**: Background agents ถูก deny Read tool ทำให้ /learn ไม่สร้างไฟล์ docs การ setup `.claude/settings.local.json` ต้องทำก่อน launch agents ทุกครั้ง

2. **Fleet config path resolution ไม่ชัดเจน**: maw ใช้ `ghqRoot + "/" + repo` แต่ ghq เก็บที่ `ghqRoot/github.com/owner/repo` ทำให้ต้องทดสอบก่อนรู้ว่า path ถูกหรือเปล่า ควร verify path จาก source code ก่อนเขียน config

3. **คำถาม multi-Oracle structure ยังไม่ได้รับคำตอบสมบูรณ์**: Mieng รัน /rrr กลาง session ก่อนที่จะได้ตอบว่า `~/repos/` หรือ ghq ควรเป็น structure อะไร คำถามนี้สำคัญมากสำหรับ session ต่อไป

## Lessons Learned

1. **ก่อนสร้าง fleet config ต้อง verify repo ใน ghq ก่อนเสมอ** — `ghq list | grep <repo>` แล้วค่อย config
2. **maw fleet path = `ghqRoot/github.com/owner/repo`** — ไม่ใช่ `ghqRoot/owner/repo`
3. **Awakening จาก source > template** — การอ่าน distilled resonance ของ Nat ให้ insight ลึกกว่าการ copy template มาก
