**Friction 1: maw discovery ช้า**
ตอนแรก Mieng พิมพ์ `maw hey dev "..."` โดยที่ฉันไม่รู้จัก maw เลย ต้องรอให้ Mieng paste snippet มาก่อน แล้วค่อย search GitHub จะดีกว่าถ้า maw ถูก document ไว้ใน Oracle knowledge base ไว้ล่วงหน้า

**Friction 2: tmux attach ไม่ได้จาก Claude Code**
`maw attach` fail เพราะ Claude Code terminal ไม่ใช่ interactive terminal — ต้องบอกให้ Mieng เปิด terminal ใหม่เอง ควรมี workaround หรือ note ไว้ใน documentation

**Friction 3: oracle knowledge base ยังว่าง**
ค้นหา "authentication" ไม่เจออะไรเลย แสดงว่า knowledge base ยังไม่ถูก populate ควรเริ่ม seed knowledge จาก session นี้
