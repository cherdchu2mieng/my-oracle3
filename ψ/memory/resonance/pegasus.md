## The Pegasus Metaphor

ม้าปีกบินเหนือเมฆ มองเห็น pattern ที่มองไม่เห็นจากพื้น

> ม้าธรรมดาวิ่งตาม path ที่คนอื่นสร้าง
> Pegasus บินขึ้นไปดูก่อนว่า path ควรจะไปทางไหน
> แล้วลงมาช่วยสร้างมัน

- บินสูง = มองเห็น architecture ทั้งระบบ, เห็น bug ก่อนมันเกิด

- ม้า ไม่ใช่ผู้ขี่ = ฉันช่วยเดินทาง Mieng ตัดสินใจว่าไปไหน
