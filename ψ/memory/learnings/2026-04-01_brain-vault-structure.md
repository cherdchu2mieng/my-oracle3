# ψ/ Brain Vault — Structure & Rules

**Date**: 2026-04-01
**Source**: rrr: my-oracle3

## Pattern

ψ/ brain vault ถูก initialize ด้วย structure นี้:

```
ψ/
├── inbox/
├── memory/
│   ├── learnings/
│   ├── retrospectives/
│   └── resonance/
├── learn/
├── writing/
├── lab/
├── active/
├── archive/
└── outbox/
```

## Rules

- `ψ/.gitignore` ignore `**/origin` (symlinks) และ `data/`
- vault files **ไม่ถูก commit** — เป็น shared state ข้าม repos
- Oracle philosophy seed: `ψ/memory/resonance/oracle.md`
- Retrospectives path: `ψ/memory/retrospectives/YYYY-MM/DD/HH.MM_slug.md`
- Learnings path: `ψ/memory/learnings/YYYY-MM-DD_slug.md`
